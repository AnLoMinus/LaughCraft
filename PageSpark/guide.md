# PageSpark 📘 — הנחיות בנייה ופריסה ל-GitHub Pages

המדריך הזה מרכז את כל מה שצריך כדי להרים אתר GitHub Pages ל-LaughCraft (עברית + RTL) ולתחזק אותו בקלות. הוא מבוסס על קבצי המאגר הקיימים (`README.md` ו-`🎓 אקדמיה לצחוקים.md`) ומגיע עם תבניות מוכנות להדבקה בתיקיית `docs/`.

## 1) מבנה מומלץ

```
docs/
├─ _config.yml
├─ assets/
│  └─ style.css
├─ index.md
├─ about.md
└─ curriculum/
   ├─ index.md
   └─ tracks.md
```

> כל הקבצים קיימים בתיקיית התבניות: `PageSpark/templates/docs/`. העתקה אחת ואתם באוויר.

## 2) שלבי הקמה (5 דקות)

1. **צור תיקיית `docs/`** בשורש המאגר.
2. **העתק את התבניות**:
   - `_config.yml`, `assets/style.css`, `index.md`, `about.md`, `curriculum/index.md`, `curriculum/tracks.md` מתוך `PageSpark/templates/docs/` אל `docs/` במאגר.
3. **פתח README לניווט** (רשות): קישור ישיר אל `/docs` יקל על צוות/קהל.
4. **הפעל GitHub Pages**:
   - Settings → Pages → Source: *Deploy from a branch*
   - Branch: `main` | Folder: `/docs`
5. **בדוק מקומית** (אופציונלי): `bundle exec jekyll serve --source docs --livereload`.

## 3) תוכן: מה כבר בפנים?

התבניות מגיעות מוכנות עם התוכן הקיים מהמאגר:

- **דף בית (`index.md`)** — הירו, ניווט מהיר, משימת פתיחה והסבר קצר למה LaughCraft.
- **מסלולי לימוד (`curriculum/index.md`)** — ארבעה מסלולים: יסודות הצחוק, סטנדאפ תודעתי, הומור אינטליגנטי, צחוק מרפא.
- **מסלולים ותרגילים (`curriculum/tracks.md`)** — LC-Method בחמישה צעדים + ארבעה קלפי תרגול (60 שניות, יומן צחוק, סטנדאפ ערכי, צחוק מרפא).
- **אודות (`about.md`)** — ליבה רעיונית, תעודת גמר, פזמון ראפ, פסוק חתימה.
- **עיצוב (`assets/style.css`)** — RTL מלא, כרטיסים, גריד, תגיות מהירות, טיפוגרפיה נקייה.

## 4) התאמות מהירות

- **צבעים/רקע**: הוסיפו ל-`:root` משתנים כמו `--bg`/`--fg` והתאימו את `.card`.
- **Header**: השתמשו ב-`layout: default` של Minima והוסיפו קישורים ב-`_includes/header.html` (אם תרצו).
- **קלפים**: השתמשו במחלקה `.card` ובגריד `.grid` כדי ליצור מקטעי "Laugh Deck".

## 5) שגרת תחזוקה

- **הוספת מסלול חדש**: הוסיפו בלוק חדש ל-`curriculum/index.md` + קלף תרגילים ב-`curriculum/tracks.md`.
- **עדכון משימת פתיחה**: ערכו את רשימת הצעדים ב-`index.md` כדי להתאים לסדנה/שבוע.
- **הרחבת יומן צחוק**: ניתן להוסיף טבלה או רשימת bullet לימים/תרגולים.

## 6) בסיס להמשך פיתוח (תבניות)

- **תיקיית התבניות** (`PageSpark/templates/docs/`) נועדה להיות בסיס: העתיקו כמו שהיא, ואז דרסו טקסטים לפי הצורך.
- לשכפול מהיר של דפי משנה, העתיקו אחד הקבצים והחליפו את `title` ואת ה-`<h1>` הראשון.
- לשינוי נראות, התאימו רק את `assets/style.css` — כך שדרוגים לא יפגעו בתוכן.

## 7) רעיונות המשך (Backlog קצר)

- הוספת **Laugh Deck**: דף קלפים עם CTA להורדה (PDF/ZIP).
- **גלריית רילז**: קטעי וידאו קצרים (embed מ-YouTube/Drive).
- **טופס הרשמה**: קישור ל-Google Form או Button להרשמה לסדנאות.
- **i18n**: גרסה מקוצרת באנגלית עם ניווט כפול (EN/HE).
